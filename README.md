# e-Distance


### Rules

1. Given `start` word and `end` word, find the shortest path between them.
2. Every intermediary on the path is a valid English word.
2. Valid operations:
   - Add letter
   - Remove letter
   - Swap letter

## Example 

`start` = 'bird'
`end` = 'plane'

Solution found in 5 steps:
'bird' -> 'bind' -> 'bine' -> 'bane' -> 'pane' -> 'plane'
